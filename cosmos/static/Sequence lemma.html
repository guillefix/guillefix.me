Conditions for relating points in the Closure of a set and existence of convergent sequences.

For a point in the closure to be guaranteed to have a convergence sequence, it is enough to be either:

  • Metrizable
  • First countable

It is easy to see that if there is a convergent sequence then the point is in the closure

The other way, for the case of metrizable is also easy to prove
