Why are there twelve notes?

video

In Pythagorean tuning, we build the notes via the two ratios 2:1 and 3:2 (or equivalently 2:1 and 3:1)

That is, we take ratios of the form (32)m2n\left(\frac{3}{2}\right)^m 2^n(​2​​3​​)​m​​2​n​​, for integers mmm and nnn. How many notes fall inside an Octave as we do this? The power of nnn just moves us between one octave and the next, so we can safely take all powers of nnn, and just focus on what happens as we increase powers of mmm. If we take the log2\log_2log​2​​ of the ratio, then the first octave lies between log2(1)=0\log_2(1)=0log​2​​(1)=0 and log2(2)=1\log_2(2)=1log​2​​(2)=1. Ratios in the first octave will lie somewhere in between. We can find these by doing log2((32)m2n) mod 1\log_2{\left(\left(\frac{3}{2}\right)^m 2^n\right)} \text{ mod } 1log​2​​((​2​​3​​)​m​​2​n​​) mod 1 =mlog2(32) mod 1=m\log_2{\left(\frac{3}{2}\right)}\text{ mod } 1=mlog​2​​(​2​​3​​) mod 1. It turns out that if we take m=0,...,11m=0,...,11m=0,...,11, we get 121212 well-spaced notes, which is what led Pythagoras and all of western music to use these 121212 notes as our chromatic scale! The fundamental reason is that log2(32)≈712\log_2{\left(\frac{3}{2}\right)}\approx \frac{7}{12}log​2​​(​2​​3​​)≈​12​​7​​, so that after adding log2(32)\log_2{\left(\frac{3}{2}\right)}log​2​​(​2​​3​​) 121212 times we are close to an integer, and thus to a beggining of an octave, that is the base note. This is the approximate period. If it was 7/12 exactly, then we would have m712 mod 1m\frac{7}{12}\text{ mod } 1m​12​​7​​ mod 1, which we can recast as 7m mod 127m \text{ mod }127m mod 12, which will visit all points 0,...,110,...,110,...,11 as 777 and 121212 are Coprime, so that 121212 iterations are needed to come back; period of periodic orbit is 121212. Therefore it can't visit the same point twice in these 121212 iterates, so that it visits 121212 points. But there are only 121212 integers here, so that it visits them all! This is fundamentally why it is so well spread out after twelve iterations.
