Ordering

A maximal element of an Partially ordered set os am element that is not smaller than any other element in the set. A minimal element of is defined dually as an element that is not greater than any other element in the set.

Often, we talk about maximal with respect to a set, in which case we often implicitly talk about the partial ordering given by the Subset relation.
