Category theory

A category C is composed of

  • a collection Ob(C) of objects: A, B, C, ...
  • a collection Ar(C) of Morphisms/arrows
  • Mappings dom,cod:Ar(C)→Ob(C)dom, cod: Ar(C) \to Ob(C)dom,cod:Ar(C)→Ob(C), specifying for each morphism which object is its domain, and which is its codomain. An arrow fff from AAA to BBB is written f:A→Bf: A \to Bf:A→B. For each such pair we define the set C(A,B)={f∈Ar(C)∣f:A→B}C(A,B) = \{f \in Ar(C) | f: A \to B\}C(A,B)={f∈Ar(C)∣f:A→B}.
