Data processing theorem#

Given a Markov chain X→Y→ZX \rightarrow Y \rightarrow ZX→Y→Z, then I(X;Y)≥I(X;Z)I(X;Y) \geq I(X;Z)I(X;Y)≥I(X;Z)

An important special case:

For any function, fff, and Random variables xxx, yyy,

I(x,y)≥I(x,f(y))I(x,y) \geq I(x, f(y))I(x,y)≥I(x,f(y))

where III is the Mutual information

A function fff for which the above inequality becomes an equality is a Sufficient statistic
