Gaussian processes are Probabilistic models, which define a probability distribution over a set of random variables (i.e. a Stochastic process); this set of random variables usually corresponds to the values of a function over a set of inputs.

Gaussian processes are usually used in Generative supervised learning. In brief, generative supervised learning works as follows: Basically assume a certain model p(y∣x)p(\mathbf{y}|\mathbf{x})p(y∣x) where the yyys correspond to the xxxs in these vectors. For prediction, we do the following: given output yyys for some inputs xxxs as data, we can have a Predictive distribution for the outputs yyys corresponding to unobserved inputs xxxs.

A Gaussian process models p(y∣x)p(\mathbf{y}|\mathbf{x})p(y∣x) as a Multivariate Gaussian distribution with a covariance matrix that is given by a Kernel function (ensuring consistency via what's called the marginaliation property). This can be interpreted as a Gaussian prior on the space of functions (i.e. a Gaussian random field)

Formally we define a Gaussian process as a Stochastic process where any finite set of Random variables in the process is jointly Gaussian distributed.

Gaussian processes with non-Gaussian likelihood

Usually the observed labels / yyy are assumed to be either equal to the function modelled by the GP, or have a Gaussian distribution around it (what's called a Gaussian likelihood – note that here the function fff works like the parameters in Bayesian inference).

If one assumes a non-Gaussian likelihood, then the problem is not Analytically tractable any more..

There are several approximations which are used then

  • Laplacian approximation
  • Expectation-propagation
  • Variational inference
