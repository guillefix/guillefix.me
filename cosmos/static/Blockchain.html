Cryptocurrency

"Trustless" auditable systems. Can reach same level of reliability/trust as for large companies/organizations but much easier, more people can benefit from this. This is particularly useful in applications where trust/reliability is important:

  • Open source software
  • Finance/markets. Systems with certain motivational/incentive structures. Connections to Game theory, Governance.
  • Industrial applications
  • etc

Also it is auditable, which can be good for some applications and not so good for others (Privacy).

Big companies, like Google, Tencent, etc are doing all these things. However, blockchain allows basically anyone to do it at the same scale, and that is the advantage.
