Although state-values suffice to define optimality, it will prove to be useful to define action-values. Given a state sss, an action aaa and a policy π\piπ, the action-value of the pair (s,a)(s,a)(s,a) under π\piπ is defined by

Qπ(s,a)=E[R∣s,a,π],Q^\pi(s,a) = E[R|s,a,\pi],\,Q​π​​(s,a)=E[R∣s,a,π],

where, now, RRR stands for the random return associated with first taking action aaa in state sss and following π\piπ thereafter.
