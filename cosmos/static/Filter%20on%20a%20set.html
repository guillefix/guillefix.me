Filter on a set

cosmos 28th June 2017 at 7:25am

A filter F\mathcal{F}F on XXX is a family of subsets of XXX such that:

(a) ∅∉F\emptyset \notin \mathcal{F}∅∉F;

(b) F\mathcal{F}F is algebraically closed under finite intersections;

(c) F\mathcal{F}F is an upper family.

An upper family refers to a family of subsets, which is an Upper set w.r.t. the Lattice of subsets of XXX, that is if a set is in the family, then any subset of that set is also in the family.
