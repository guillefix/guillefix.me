If Group acts on a set SSS, it will also act on the set of all subsets (Power set).

Definition. Basically g⋅U={g⋅u∣u∈U}g\cdot U = \{ g\cdot u | u \in U\}g⋅U={g⋅u∣u∈U} (the set obtained by aplying ggg to the elements of the subset). This defines a Group action

Can also restrict action to the set of subsets of a given cardinality. This is because the group action is always an injection (as can be seen by thinking of inverses), so it preserves set cardinality.
