Baum-Welch algorithm

cosmos 9th January 2017 at 11:38am
Hidden Markov model
  • From θ\thetaθ calculate hidden states probabilities.
    • Hidden state probability
    • Hidden state transition probability
  • From hidden states probabilities calculate transition and emission counts estimates
  • From counts estimates θ\thetaθ
    • Transition matrix
    • Emission probabilities
    • Initial probability
  • repeat, stop if increment in likelihood below threshold.
