A type of Relation between two Sets, such that for each element xxx belonging to the set called domain (aka input set), there is a unique element yyy belonging to the set called co-domain (aka output set, or range). We denote this notationally as x↦yx \mapsto yx↦y. We typically accompany this notation, with a notation declaring what are the domain and co-domain of the function. We denote that the function fff is between the domain X\mathcal{X}X and the co-domain Y\mathcal{Y}Y by:

f:X→Yf: \mathcal{X} \to \mathcal{Y}f:X→Y

Types of function

  • Injective function
  • Surjective function
  • Bijection or bijective function.
