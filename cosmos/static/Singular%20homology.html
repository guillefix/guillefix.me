Homology

Definition. For a Topological space XXX, we consider the set of all Singular simplexes, the corresponding Chain group, the Boundary maps (vid) between them, giving rise to a Chain complex (called Singular chain complex). Then singular homology is defined as the Singular homology groups, which are the quotient of the kernel of the outgoing map by the image of the incoming map (that is the Homology groups).

Note that the set of singular simplices (continuous functions from simplex to space) is most often uncountably Infinite, and so the Chain groups are also infinite. However, the quotient groups defining the singular homology turn out to be finite. (see here).

Intuition

Homology is functorial

If we have a Continuous function between topological spaces, this gives us a Group homomorphism between the corresponding homology groups. This will mean that Homology is a Topological invariant, and so can be used to study if two spaces are homeomorphic.

Formal definition of induced Chain map f#f_\#f​#​​, defined as the pushforward of singular simplex, and extending linearly, and it is easy to check directly that it commutes with boundary map (by using its definition), so it is indeed a chain map.

This chain map induces a map on homology (see below also), defined by lifting f_* to the homology classes, and it is called the pushforward by fff.
