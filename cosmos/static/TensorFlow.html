In tensor flow, you:

  1. Define a computational graph, that manipulates tensors, and data. parts of the computational graph
    1. Placeholder variables used to change the input to the graph. x = tf.placeholder(tf.float32, [dimensions]). Note None can be used as a dimension, that can take any value (calculated from input data..)
    2. Model variables that are going to be optimized so as to make the model perform better. weights = tf.Variable(initializationValues)
    3. The model which is essentially just a mathematical function that calculates some output given the input in the placeholder variables and the model variables. Example@ logits = tf.matmul(x, weights) + biases
    4. A cost function that can be used to guide the optimization of the variables. Example: cross_entropy = tf.nn.softmax_cross_entropy_with_logits(logits=logits, labels=y_true), cost = tf.reduce_mean(cross_entropy)
    5. An optimization method which updates the variables of the model. Example: optimizer = tf.train.GradientDescentOptimizer(learning_rate=0.5).minimize(cost)
  2. Begin a session (tf.Session()), and train, and run the computational graph, to get its result.
    1. session.run(tf.global_variables_initializer())
    2. session.run(optimizer, feed_dict=feed_dict_train), where the keys in the feed_dict should match the placeholder variables.
