Simplicial cycle

cosmos 15th October 2017 at 9:55pm

A path through the edges of a simplicial complex, or a Delta-complex, where the path returns to the same vertex.

When drawing the path, if we draw a part of a path, and then we draw the same part but backwards, then they "cancel out", meaning, that the path is the same as removing those parts..

There are higher dimensional analogues.
