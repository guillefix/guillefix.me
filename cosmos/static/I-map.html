Graphical model

An I-map (independence map) for a Probability distribution PPP is any Graphical model GGG such that the set of independences implied by the network (I(G)I(G)I(G)) is a subset of the set of independences of PPP (I(P)I(P)I(P)) (see here), i.e. I(G)⊂I(P)I(G) \subset I(P)I(G)⊂I(P)

A perfect (independence) map is one such that I(G)=I(P)I(G) = I(P)I(G)=I(P)
