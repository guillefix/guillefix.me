Ordering

A partial ordering on a Set XXX is a (binary) Relation on XXX, ⪯\preceq⪯ that is:

  • reflexive: for all x∈X,x⪯xx \in X, x \preceq xx∈X,x⪯x.
  • antisymmetric: for all x,y∈Xx, y \in Xx,y∈X, if x⪯yx \preceq yx⪯y, and y⪯xy \preceq xy⪯x, then x=yx=yx=y.
  • transitive: for all x,y,z∈Xx, y, z \in Xx,y,z∈X, if x⪯yx \preceq yx⪯y and y⪯zy \preceq zy⪯z, then x⪯z x \preceq zx⪯z/

A set with a partial ordering is called a Partially ordered set (or poset).

A Pre-order is a weaker kind of relation

For many common examples, the Partial ordering ⪯\preceq⪯ is often interpreted as ≤\leq≤ (or less than or equal, for Real numbers).
