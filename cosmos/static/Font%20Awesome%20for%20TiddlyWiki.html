Font Awesome for TiddlyWiki

3rd February 2018 at 1:54pm

You can add Font Awesome to your wiki by using one of the sources below.

@tobibeer created a great reference page for how to employ Font Awesome with TiddlyWiki. I recommend you follow his instructions on how to install and use Font Awesome: http://tobibeer.github.io/tw/fa.

@TheDiveO also created a Font Awesome plugin. See TW5FontAwesome Plugin. However, his provided version is out of date.
