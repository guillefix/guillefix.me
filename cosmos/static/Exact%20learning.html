Computational learning theory

Learn a DFA representing a target language L⊆Σ∗L \subseteq \Sigma^*L⊆Σ​∗​​ using membership and equivalence queries to a teacher

  • Membership query. selects string and asks whether that string is part of the langauge
  • Equivalence query. learner selects hypothesis DFA H and asks does L(H)=LL(H)=LL(H)=L? If yes, the algo halts. If no, the teacher gives a counterexample.

We want to exactly learn L using # of queries polynomial in the size of a in DFA for L and the length of the longer coutnerexample given by the teacher.
