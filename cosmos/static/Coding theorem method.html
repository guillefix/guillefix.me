Using the coding theorem to estimate the Kolmogorov complexity of short strings. The estimate is defined as:

Km(s)=−log2(D(n,k)(s))K_m (s) = -\log_2(D(n,k)(s))K​m​​(s)=−log​2​​(D(n,k)(s))

where

D(n,k)(s):=∣{T∈(n,k):T produces s}∣∣{T∈(n,k):T halts}∣D(n,k)(s) : = \frac{|\{T \in (n,k) : T \text{ produces } s\}|}{|\{T \in (n,k) : T \text{ halts}\}|}D(n,k)(s):=​∣{T∈(n,k):T halts}∣​​∣{T∈(n,k):T produces s}∣​​

where (n,k)(n,k)(n,k) is the set of Turing machines with nnn states and kkk letters in the alphabet of the input tape. The Turing machines are fed a blank tape, and whether the program halts is determined using a Busy beaver function.
