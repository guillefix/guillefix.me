Tiling

A tiling is said to be kkk-anisohedral if it is not nnn-isohedral for any n<kn < kn<k. A tiling is called anisohedral if it is 222-anisohedral, that is if it is not isohedral.

A Tile which only allows anisohedral tilings, so no isohedral tilings, is called an anisohedral tile
