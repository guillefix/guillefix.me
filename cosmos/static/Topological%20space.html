Topological space

guillefix 14th July 2016 at 3:52am

A topological space is a Set XXX, with a collection of distinguished Subsets called Open sets, called the topology of the set. These must satisfy:

  • the union of an arbitrary collection of open sets is open;
  • the intersection of any finite collection of open sets is open;
  • The empty set ∅\emptyset∅ and the whole set, XXX itself are both open.

It can also be shown that: A neighbourhood space (X,N)(X, \mathcal{N})(X,N) is a topological space if and only if each Filter N(x)\mathcal{N}(x)N(x) has a Filter base consisting of Open sets.

Remark: for family CCC of subsets of a set XXX, there exists a unique 'smallest' topology on XXX for which CCC is a subbase: namely that topology whose open sets are defined to be all arbitrary unions of the collection of all finite intersections of elements of CCC.

Connections with lattices

The set of open sets in a topology forms a lattice, where the partial ordering is set inclusion. Also the set of topologies on a set XXX can also be equipped with a natural lattice structure.
