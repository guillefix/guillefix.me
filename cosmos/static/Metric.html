A metric on a Set XXX is a map d:X×X→Rd: X \times X \rightarrow \mathbb{R}d:X×X→R (i.e. from the Cartesian square of XXX to the Real numbers), that satisfies the conditions:

  • symmetry: d(x,y)=d(y,x)d(x,y) = d(y,x)d(x,y)=d(y,x)
  • positivity: d(x,y)≥0d(x,y) \geq 0d(x,y)≥0, and =0=0=0 if, and only if, x=yx=yx=y.
  • Triangle inequality: d(x,y)≤d(x,z)+d(z,y)d(x,y) \leq d(x,z) + d(z,y)d(x,y)≤d(x,z)+d(z,y).
