Aside: a node with a "high" degree is sometimes called a 'hub'.
A network where all nodes have same degree is called 'regular'.

A network is sparse if ρ→0\rho \rightarrow 0ρ→0 as n→∞n \rightarrow \inftyn→∞. It is dense otherwise. These definitions make sense mathematically when one has a model for an ensemble of graphs, that can be defined for any nnn. For an empirical network, one has to situations:

  • One has to find an appropriate model to define an ensemble of random graphs for different values of nnn, that somehow captures the type of network of the empirical one.

Directed networks

For directed networks one has two types of degree:

in-degree, the number of ingoing edges (sum of a row in adj. matrix)

out-degree, the number of outgoing edges (sum of columns in adj. matrix).

Now the total number of edges mmm is:

m=∑j=1nkjin=∑j=1nkjout=∑ijAijm=\sum_{j=1}^n k_j^{\text{in}}=\sum_{j=1}^n k_j^{\text{out}}=\sum_{ij}A_{ij}m=∑​j=1​n​​k​j​in​​=∑​j=1​n​​k​j​out​​=∑​ij​​A​ij​​

as each edge has one ingoing end and one outgoing end. Clearly then the mean degrees are equal: cin=cout≡c=mnc_{\text{in}}=c_{\text{out}}\equiv c=\frac{m}{n}c​in​​=c​out​​≡c=​n​​m​​.
